# frenchbench-app
FrenchBench intelligent team builder App and API using Node.js, React.js, Next.js, MySQL


## Installation

```
npm i
```

## Configuration

Copy `.env.development` as `.env` and edit.

## Build

```
npm run build
```

## Run

```
npm run start
```

## Disable telemetry

```
npx next telemetry disable
```
